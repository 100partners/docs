  • lowercase for display, headings, ui labels, links
  • sentence case in blog body copy
  • letter-spacing 0.04–0.14em on small caps
  • use monospace everywhere — no exceptions
// don't
  • introduce a sans-serif for "variety"
  • use uppercase (unless it's inside the logo)
  • mix bold and italic in display type
  • use em dashes (—). ever.

04 · [spacing] the 4-8-16 scale

vertical rhythm is aggressive. whitespace IS the design — it's what makes the minimal terminal aesthetic feel editorial instead of austere.

4px · hairline
8px · tight
14px · card gap
18px · para break
24px · card padding
28px · page inset (mobile)
40px · wrap top/bottom
56px · major section
72px · sec-head margin-top
120px · wrap bottom (desktop)

content width

  • homepage / landing: 820px max
  • blog post: 720px max
  • mobile inset: 16px
  • desktop inset: 28px

borders & rules

  • card/block: 1px solid ink
  • section rule: 1px solid soft
  • soft divider: 1px dotted soft
  • accent divider: 2–3px solid accent (left-edge only)
  • radius: 0 (flat corners everywhere except avatars)
  • shadow: none

07 · [voice] how we sound

100 partners talks like an operator who's been in the trenches, not a marketer reading a script. short sentences. contrarian angles. specific numbers over adjectives.

// voice
  • lowercase in display & ui
  • direct, declarative, contrarian
  • name real companies, real numbers, real outcomes
  • use // comment prefix for meta lines
  • use [bracketed labels] for category tags
"master the most unobvious sales channel sitting in plain sight."
// anti-voice
  • no em dashes (—) — ever
  • no "revolutionary" / "game-changing" / "synergies"
  • no exclamation marks outside of transactional emails
  • no emojis in body copy (ok sparingly in newsletter)
  • no corporate hedge-speak
"revolutionize your growth with our cutting-edge partner ecosystem solutions."
// signature moves
  • open with the contrarian claim
  • prove it with a named customer + number
  • end with the one-line CTA in lowercase
  • blinking cursor after display headlines
  • terminal scan for comparison or status
// house phrases: "partner program installed in 90 days" · "the most unobvious sales channel" · "ai commoditized content. trust is what's left." · "let's partner."
